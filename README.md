iisns - 地球村入口
------------------

**iiSNS 是基于 yii2 的 SNS 社区系统，一站式解决社区建站。可以写文章，做记录，上传图片，论坛聊天等。还可以用来做内容管理系统（CMS）。iiSNS 是一个免费的开源项目，在 MIT 许可证下授权发布。**



###您可以用于

- 粉丝营销社区
- 新媒体行业
- 企业文化建设

更多用途，等你来发掘。演示:[www.iisns.com](http://www.iisns.com)
